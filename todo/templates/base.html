About this site

Hi, I'm Dave!

DMF Todo is a simple CRUD app, robustly deployed.

DMF Todo is built with Django, using Django templates styled with Bootstrap.
Data is stored using PostgreSQL.
Emails are served by Amazon SES.

DMF Todo is deployed on an AWS EC2 instance, with Gunicorn as the application server.
This stack is orchestarted using Docker. Nginx is configured as a reverse proxy, serves static files for this and other sites on the instance, and logs all traffic.
Namecheap is the domain registrar, and LetsEncrypt provides the TLS certificate.

This site's code is viewable on my github: https://github.com/davefriedman/todo.
